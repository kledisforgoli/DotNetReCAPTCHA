# DotNetReCAPTCHA

This is a Google ReCAPTCHA implementation in .NET Core. The user should complete the ReCAPTCHA when logging in or signing up. Furthermore, all inputs are validated. 
I've also added a session expiry funcionality that logs a user out after a couple of seconds.

## Tech Stack
- .NET Core
- Entity Framework Core
- SQL Server
- Bootstrap
- HTML + CSS
- Google ReCAPTCHA
